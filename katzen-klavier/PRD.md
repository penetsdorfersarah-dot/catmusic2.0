# PRD — Katzen-Klavier-App

## Produkt-Vision
Eine Browser-App bei der man über die Laptop-Tastatur ein Klavier spielt — jede Note erzeugt einen KI-generierten Katzengeräusch-Sound via ElevenLabs.

## Problem
Normale Online-Klaviere sind langweilig. Es gibt keine spaßige, niedliche Alternative die gleichzeitig KI-Audio-Technologie nutzt.

## Zielgruppe
- Kinder & Eltern die zusammen spielen
- Erwachsene die Spaß wollen
- Entwickler die ElevenLabs ausprobieren wollen

## Features (MVP)
- [ ] Visuelles Klavier-Interface (2 Oktaven, 29 Tasten)
- [ ] Laptop-Tastatur Steuerung (A-Z + Sonderzeichen)
- [ ] Web Audio API Töne (Phase 1)
- [ ] ElevenLabs Katzengeräusche (Phase 2)
- [ ] Visuelle Key-Press Animationen

## Features (Nice-to-have)
- [ ] Record & Playback
- [ ] Verschiedene Katzen-Stimmen
- [ ] Mobile Touch-Support
- [ ] Sustain-Pedal (Leertaste)
- [ ] Export als Audio-Datei

## Tech Stack
- Vanilla HTML/CSS/JS (kein Framework)
- Web Audio API
- ElevenLabs Sound Generation API

## Tastatur-Mapping
- Weiße Tasten: A S D F G H J K L ; ' Enter
- Schwarze Tasten: W E T Y U O P [ ] \

## Erfolgs-Metriken
- App lädt in unter 2 Sekunden
- Audio-Latenz unter 100ms
- Alle 29 Tasten spielbar

## Phasen
1. Phase 1 — Basis-Klavier mit Web Audio API
2. Phase 2 — ElevenLabs Integration
3. Phase 3 — Polish, Animationen, Extras
